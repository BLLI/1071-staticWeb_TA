### CH5 表格基本元素
* `<table> ` : 用來標記表格
* `<tr> ` : 用來在表個中標示**一列(ROW)**
* `<td> ` : 用來在一列中標示**儲存格(COL)**
* `<th> ` : 用來在一列中標示**標題**儲存格，其內容會**置中**並加上**粗體**
  ### 表格基本框架
  ```
  <table border="1">
    <tr>
      <td>1-1</td>
      <td>1-2</td>
     </tr>
     <tr>
      <td>2-1</td>
      <td>2-2</td>
     </tr>
     <tr>
      <td>3-1</td>
      <td>3-2</td>
    </tr>
  </table>
  ```
  |第一欄|第二欄|
  |-----|-----|
  | 1-1 | 1-2 |
  | 2-1 | 2-2 |
  | 3-1 | 3-2 |
